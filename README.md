# Условия участия

В репе содержится код реализации внешний сортировки. Написано новичком и, очевидно, можно лучше. Требуется предоставить свой вариант рефакторинга кода, не меняя его функционала.

Для этого нужно:
* Форкнуть репу
* Отрефакторить на свой вкус
* Создать PR
